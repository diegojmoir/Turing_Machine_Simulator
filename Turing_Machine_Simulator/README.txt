So first you have to select a turing machine, then you have to put the input and click 
on the button. 
Alphabets:

Add machine: {1,+,B}
Sub machine: {1,-,B}
Mult machine: {1,*,=,B} You have to put "=" at the end of the input
Copy machine: {a,b,c,X,Y,Z,?,B}
Palindrome machine: {a,b,c,B}
